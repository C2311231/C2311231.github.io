SkySpindle

Welcome to SkySpindle

A lightweight and stylish CNC control program made to mimic the appearance of Mainsail but for grbl.

Features

  • Webinterface: SkySpindle interface is hosted on a webserver so access it anywhere on the local network!
  • Easy File Uploading and Editing: Simply drag and drop files to be added and select any file to show in the editor with syntax highlighting!
  • Line Gcode progress: The editor shows a indicator on the last line sent to the CNC machine.
  • Live Console Access: See what is being sent to and recived from the machine at anytime!
  • No Command Limitations: Any typeable command can be sent to the machine at anytime!
  • Emergency Stop: Easy to access emergency stop button restarts the controler instantly and will come up in an alarm state if the machine is in motion.
